
# Types of Variables

In statistics, a **variable** is any characteristic that can take different values for different individuals or items.  
Variables are classified into two main types: **Quantitative** and **Qualitative**.  
Each type behaves differently and requires different methods of analysis.

A **qualitative** or **categorical variable** does **not** represent numerical values.
