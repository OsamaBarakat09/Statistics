## ğŸ“˜ Inferential Statistics

Inferential statistics use **sample data** to make conclusions or predictions about a **larger population**.

### What inferential statistics include:
- Estimating population values  
- Predicting future outcomes  
- Hypothesis testing  
- Confidence intervals  
- Regression analysis  

### Example  
A company tests **50 light bulbs** to estimate the average lifespan of all bulbs it manufactures.  
From this sample, they can estimate the lifespan of millions of bulbs.

---

## ğŸ“Œ Population

## ğŸ“Œ Sample

## ğŸ“Œ Census

**Example:** National population census.

## ğŸ“Œ Sample Survey  
Data collected from **part** of the population.  
**Example:** Asking 500 customers about their experience.

## ğŸ“Œ Representative Sample

## ğŸ“Œ Random Sample

## ğŸ“Œ Element / Member

## ğŸ“Œ Variable

## ğŸ“Œ Observation

## ğŸ“Œ Data Set

# 3ï¸âƒ£ Types of Variables

Variables determine how data can be measured and analyzed.

---

## 1. Quantitative Variables  
These variables can be measured numerically.

### Examples:
- Salary  
- Height  
- Weight  
- Temperature  

### ğŸ”¹ Discrete Variables  
Values are **countable** and cannot take decimals.  
**Examples:**  
- Number of students  
- Number of cars  
- Number of products sold  

### ğŸ”¹ Continuous Variables  
Values can take **any number**, including decimals.  
**Examples:**  
- Weight (70.5 kg)  
- Height (172.3 cm)  
- Time (6.27 seconds)  

---

## 2. Qualitative (Categorical) Variables  
Variables that represent categories, not numbers.

### Examples:
- Gender (Male / Female)  
- Country (Italy / Egypt)  
- Department (HR / Finance / Marketing)  

These cannot be added or averaged because they are **not numeric**.
